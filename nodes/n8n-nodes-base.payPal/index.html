# Example Usage

This workflow shows you how to create a PayPal batch payout. You can also find the workflow on this website. This example usage workflow uses the following two nodes.

# 1. Start node

# 2. PayPal node

  1. First of all, you'll have to enter credentials for the PayPal node. You can find out how to do that here.
  2. Enter an ID in the Sender Batch ID field.
  3. Click on Execute Node to run the workflow.
